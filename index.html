Overview

A Python-based trading portfolio simulator that lets you simulate trading using fictional market data while tracking holdings, cash balance, and daily performance.

The app includes core functions such as buying and selling assets, editing holdings, viewing simulated market prices, and generating performance reports.

Key Features

  • Save & Load Portfolios — Portfolio data is persistently stored and retrieved using JSON file handling, allowing sessions to be saved and resumed at any time.
  • Simulated Market Updates — Prices update dynamically each trading day within a ±5% range, replicating real-world market volatility in a controlled environment.
  • Transaction History & Daily Summaries — Every buy and sell action is logged, with daily summaries generated to provide a clear overview of portfolio performance over time.

Challenges & Learnings

This project strengthened my Python fundamentals, particularly in data structuring and file management. It also gave me a better appreciation for how real trading systems handle dynamic data, and is something I plan to expand with a graphical interface and real market data in the future.
